## 一、项目概况

**IntervYOU** — AI 面试教练平台。面向校招生，覆盖简历诊断 → 模拟面试 → 复盘报告 → 优化简历的完整闭环。

**技术栈**：Next.js 14 + TypeScript + Prisma/SQLite + Tailwind CSS + DeepSeek V3（可切换 Claude/OpenAI）

## 三、技术亮点

1. **模块化Prompt系统** — 岗位自动检测、四层递进、三模式差异化，全部由TypeScript函数组装
2. **历史脱敏** — Agent评分JSON不回传给LLM，避免评分泄露和锚定偏差
3. **分级降级** — JSON解析失败→重试→提取纯文本→上下文默认问题，四层Fallback
4. **多Provider抽象** — 一套代码兼容DeepSeek/Claude/OpenAI，用户可自带Key
5. **服务端防重复** — 前15字匹配检测，自动替换为点名具体经历的换题语

## 四、待改进项

### P0 — 面试体验核心
- [ ] DeepSeek JSON格式偶发不稳定 → 考虑切换到Claude（成本高20倍）或加更多降级逻辑
- [ ] 四层递进偶尔漂移（从第3层回退到第2层） → 在prompt中更明确标注当前层级
- [ ] 综合模拟模式缺少真实阶段切换感 → 可考虑在metadata中强制阶段推进

### P1 — 功能完善
- [ ] 无真正的流式输出（当前全量等待） → SSE或WebSocket
- [ ] 无面试录音回放分析
- [ ] 无简历对比功能
- [ ] 无多语言支持

### P2 — 体验打磨
- [ ] 移动端面试页输入体验需优化
- [ ] 无暗色模式
- [ ] 诊断评分偶有集中趋势（60-75区间）
- [ ] 复盘报告无打印样式
